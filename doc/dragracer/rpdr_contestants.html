rpdr_contestantsR Documentation

RuPaul's Drag Race Contestant Data

Description

These are contestant-level data for RuPaul's Drag Race for all available seasons (currently through Season 14). Data include all sorts of information.

Usage

rpdr_contestants

Format

A data frame with the following 5 variables.

season

a character vector for season

contestant

a character vector for contestant's name, by season

age

a numeric vector for the respondent's purported age as of filming

dob

a date vector for the contestant's purported date of birth

hometown

a character vector for hometown.

Details

Note: Cynthia Lee Fontaine, Shangela, Eureka O'Hara, and Vanessa Vanjie Mateo will appear in this data set twice.
