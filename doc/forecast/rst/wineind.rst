======= ===============
wineind R Documentation
======= ===============

Australian total wine sales
---------------------------

Description
~~~~~~~~~~~

Australian total wine sales by wine makers in bottles <= 1 litre. Jan
1980 – Aug 1994.

Usage
~~~~~

.. code:: R

   wineind

Format
~~~~~~

Time series data

Source
~~~~~~

Time Series Data Library. https://pkg.yangzhuoranyang.com/tsdl/

Examples
~~~~~~~~

.. code:: R

   tsdisplay(wineind)
