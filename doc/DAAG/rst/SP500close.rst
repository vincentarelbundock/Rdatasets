========== ===============
SP500close R Documentation
========== ===============

Closing Numbers for S and P 500 Index
-------------------------------------

Description
~~~~~~~~~~~

Closing numbers for S and P 500 Index, Jan. 1, 1990 through early 2000.

Usage
~~~~~

::

   SP500close

Source
~~~~~~

Derived from SP500 in the MASS library.

Examples
~~~~~~~~

::

   ts.plot(SP500close)
