      ======= ===============
      biomass R Documentation
      ======= ===============

      .. rubric:: Biomass Data
         :name: biomass-data

      .. rubric:: Description
         :name: description

      The ``biomass`` data frame has 135 rows and 8 columns. The
      ``rainforest`` data frame is a subset of this one.

      .. rubric:: Usage
         :name: usage

      ::

         biomass

      .. rubric:: Format
         :name: format

      This data frame contains the following columns:

      dbh
         a numeric vector

      wood
         a numeric vector

      bark
         a numeric vector

      fac26
         a factor with 3 levels

      root
         a numeric vector

      rootsk
         a numeric vector

      branch
         a numeric vector

      species
         a factor with levels ``Acacia mabellae``, ``C. fraseri``,
         ``Acmena smithii``, ``B. myrtifolia``

      .. rubric:: Source
         :name: source
