=================== ===============
state_abb_character R Documentation
=================== ===============

US State Facts and Figures - U.S. State Abbreviations
-----------------------------------------------------

Description
~~~~~~~~~~~

The dataset name has been changed to 'state_abb_character' to avoid
confusion with other packages in the R ecosystem. This naming convention
helps distinguish this dataset as part of the 'usdatasets' package and
identifies it as a character vector. The original content of the dataset
has not been modified.

Usage
~~~~~

.. code:: R

   data(state_abb_character)

Format
~~~~~~

A character vector with 50 elements representing U.S. state
abbreviations:

state_abb
   Character vector of state abbreviations, e.g., "AL" for Alabama, "CA"
   for California.

Source
~~~~~~

U.S. state abbreviations.
