      .. rubric:: Usage

      .. rubric:: Format
         :name: format

      A dataframe containing :

      docno
         doc no.

      weeks
         weeks

      numbids
         count

      takeover
         delta (1 if taken over)

      bidprem
         bid Premium

      insthold
         institutional holdings

      size
         size measured in billions

      leglrest
         legal restructuring

      rearest
         real restructuring

      finrest
         financial restructuring

      regulatn
         regulation

      whtknght
         white knight
