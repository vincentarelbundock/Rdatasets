Economic Journals Data Set
--------------------------

Description
~~~~~~~~~~~

a cross-section from 2000

*number of observations* : 180

*observation* : goods

Usage
~~~~~

::

   data(Journals)

Format
~~~~~~

A dataframe containing :

title
   journal title

pub
   publisher

society
   scholarly society ?

libprice
   library subscription price

pages
   number of pages

charpp
   characters per page

citestot
   total number of citations

date1
   year journal was founded

oclc
   number of library subscriptions

field
   field description

Source
~~~~~~

Professor Theodore Bergstrom of the Department of Economics at the
University of California, San Diego.

References
~~~~~~~~~~

Stock, James H. and Mark W. Watson (2003) *Introduction to
Econometrics*, Addison-Wesley Educational Publishers, chapter 6.
