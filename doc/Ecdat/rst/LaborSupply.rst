   =========== ===============
   LaborSupply R Documentation
   =========== ===============

   .. rubric:: Wages and Hours Worked
      :name: LaborSupply

   .. rubric:: Description
      :name: description

   a panel of 532 observations from 1979 to 1988

   *number of observations* : 5320

   .. rubric:: Usage
      :name: usage

   ::

      data(LaborSupply)

   .. rubric:: Format
      :name: format

   A dataframe containing :

   lnhr
      log of annual hours worked

   lnwg
      log of hourly wage

   kids
      number of children

   age
      age

   disab
      bad health

   id
      id

   year
      year
