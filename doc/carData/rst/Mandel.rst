      .. rubric:: Description
         :name: description

      The ``Mandel`` data frame has 8 rows and 3 columns.

      .. rubric:: Usage
         :name: usage

      ::

         Mandel

      .. rubric:: Format
         :name: format

      This data frame contains the following columns:

      x1
         first predictor.

      x2
         second predictor.

      y
         response.
