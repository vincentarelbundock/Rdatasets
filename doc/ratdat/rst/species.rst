      ======= ===============
      species R Documentation
      ======= ===============

      .. rubric:: Species data.
         :name: species-data.

      .. rubric:: Description
         :name: description

      Data on species captured at the Portal Project

      .. rubric:: Usage
         :name: usage

      .. code:: R

         species

      .. rubric:: Format
         :name: format

      A data frame with columns:

      species_id
         Species identifier

      genus
         The genus of the species

      species
         The latin species name

      taxa
         General taxonomic category of the species
