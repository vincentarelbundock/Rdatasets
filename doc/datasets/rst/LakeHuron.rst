      .. rubric:: Description
         :name: description

      1875–1972.

      .. rubric:: Usage
         :name: usage

      ::

         LakeHuron

      .. rubric:: Format
         :name: format

      A time series of length 98.

      Series and Forecasting*. Springer, New York. Sections 5.1 and 7.6.
