Household data
--------------

Description
~~~~~~~~~~~

This dataset is based on an example in
``vignette("datatable-reshape", package = "data.table")``

Usage
~~~~~

.. code:: R

   household

Format
~~~~~~

A data frame with 5 rows and 5 columns:

family
   Family identifier

dob_child1
   Date of birth of first child

dob_child2
   Date of birth of second child

name_child1
   Name of first child

?

name_child2
   Name of second child
