OpenIntro colors
----------------

Description
~~~~~~~~~~~

A character string of full colors from ``IMSCOL[,1]``

Usage
~~~~~

.. code:: R

   openintro_colors

Format
~~~~~~

A named character string with 9 elements: "blue", "green", "pink",
"yellow", "red", "black", "gray", "lgray

Examples
~~~~~~~~

.. code:: R


   openintro_colors

   openintro_colors["blue"]
