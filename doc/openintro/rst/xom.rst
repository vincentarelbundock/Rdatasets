Exxon Mobile stock data
-----------------------

Description
~~~~~~~~~~~

Monthly data covering 2006 through early 2014.

Usage
~~~~~

::

   xom

Format
~~~~~~

A data frame with 98 observations on the following 7 variables.

date
   Date.

open
   a numeric vector

high
   a numeric vector

low
   a numeric vector

close
   a numeric vector

volume
   a numeric vector

adj_close
   a numeric vector

Source
~~~~~~

Yahoo! Finance.

Examples
~~~~~~~~

::


   xom
