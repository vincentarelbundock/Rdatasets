OpenIntro palettes
------------------

Description
~~~~~
A list with OpenIntro color palettes

Usage
~~
.. code:: R

   openintro_palettes

Format
~~~

A list with 8 color palettes: main, two, three, four, five, cool, hot,
gray

Examples
~~~~

.. code:: R


   openintro_palettes

   openintro_palettes$main
   openintro_palettes$three
   openintro_palettes$cool
   openintro_palettes$hot
