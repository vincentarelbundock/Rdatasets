      :name: unemploy_pres

   .. rubric:: Description
      :name: description

   Covers midterm elections.

   .. rubric:: Usage
      :name: usage

   .. code:: R

      unemploy_pres

   .. rubric:: Format
      :name: format

   A data frame with 29 observations on the following 5 variables.

   year
      Year.

   potus
      The president in office.

   party
      President's party.

   unemp
      Unemployment rate.

   change
      Change in House seats for the president's party.
