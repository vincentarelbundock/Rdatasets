Simulated polling data set
--------------------------

Description
~~~~~~~~~~~

Simulated data for a fake political candidate.

Usage
~~~~~

::

   toohey

Format
~~~~~~

A data frame with 500 observations on the following variable.

vote_for
   a factor with levels ``no`` ``yes``

Examples
~~~~~~~~

::


   toohey
