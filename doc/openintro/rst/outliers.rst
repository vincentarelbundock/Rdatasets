======== ===============
outliers R Documentation
======== ===============

Simulated data sets for different types of outliers
---------------------------------------------------

Description
~~~~~~~~~~~

Data sets for showing different types of outliers

Usage
~~~~~

::

   outliers

Format
~~~~~~

A data frame with 50 observations on the following 5 variables.

x
   a numeric vector

y
   a numeric vector

x_inf
   a numeric vector

y_lev
   a numeric vector

y_out
   a numeric vector

Examples
~~~~~~~~

::


   outliers
