===== ===============
books R Documentation
===== ===============

Sample of books on a shelf
--------------------------

Description
~~~~~~~~~~~

Simulated data set.

Usage
~~~~~

::

   books

Format
~~~~~~

A data frame with 95 observations on the following 2 variables.

type
   a factor with levels ``fiction`` and ``nonfiction``

format
   a factor with levels ``hardcover`` and ``paperback``

Examples
~~~~~~~~

::


   table(books)
