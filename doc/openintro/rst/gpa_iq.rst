   .. rubric:: Description
      :name: description

   Data on 78 students including GPA, IQ, and gender.

   .. rubric:: Usage
      :name: usage

   ::

      gpa_iq

   .. rubric:: Format
      :name: format

   A data frame with 78 observations representing students on the
   following 5 variables.

   obs
      a numeric vector

   gpa
      Grade point average (GPA).

   iq
      IQ.

   gender
      Gender.

   concept
      a numeric vector
