Subject Level Analysis Dataset
------------------------------

Description
~~~~~
An example subject level analysis dataset

Usage
~~
.. code:: R

   admiral_adsl

Format
~~~

An object of class ``tbl_df`` (inherits from ``tbl``, ``data.frame``)
with 306 rows and 54 columns.

Source
~~~

Derived from the ``dm`` and ``ds`` datasets using ``{admiral}``
(https://github.com/pharmaverse/admiral/blob/main/inst/templates/ad_adsl.R)

See Also
~~~~

Other datasets: ``admiral_adlb``, ``example_qs``, ``queries``,
``queries_mh``
