   .. rubric:: NHEFS Codebook
      :name: nhefs_codebook

   .. rubric:: Description
      :name: description

   ``nhefs_codebook`` is the codebook for ``nhefs`` and
   ``nhefs_complete``.

   .. rubric:: Usage
      :name: usage

   .. code:: R

      nhefs_codebook

   .. rubric:: Format
      :name: format

   A data frame with 64 rows and 2 variables.

   variable
      The variable being described

   description
      The variable description
