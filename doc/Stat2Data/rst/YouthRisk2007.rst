   ============= ===============
   YouthRisk2007 R Documentation
   ============= ===============

   .. rubric:: Riding with a Driver Who Has Been Drinking
      :name: YouthRisk2007

   .. rubric:: Description
      :name: description

   Risky behavior (riding with a drinking driver) in youths

   .. rubric:: Format
      :name: format

   A dataset with 13387 observations on the following 6 variables.

   +---------------------+-----------------------------------------------+
   | ``ride.alc.driver`` | ``1``\ =rode with a drinking driver in past   |
   |                     | 30 days or ``0``\ =did not                    |
   +---------------------+-----------------------------------------------+
   | ``female``          | ``1``\ =female or ``0``\ =male                |
   +---------------------+-----------------------------------------------+
   | ``grade``           | Year in high school: ``9``, ``10``, ``11``,   |
   |                     | or ``12``                                     |
   +---------------------+-----------------------------------------------+
   | ``age4``            | Age (in years)                                |
   +---------------------+-----------------------------------------------+
   | ``smoke``           | Ever smoked? ``1``\ =yes or ``0``\ =no        |
   +---------------------+-----------------------------------------------+
   | ``DriverLicense``   | Have a driver's license? ``1``\ =yes or       |
   |                     | ``0``\ =no                                    |
   +---------------------+-----------------------------------------------+
   |                     |                                               |
   +---------------------+-----------------------------------------------+

   .. rubric:: Details
      :name: details

   This dataset is derived from the 2007 Youth Risk Behavior
   Surveillance System (YRBSS), which is an annual survey conducted by
   the Centers for Disease Control and Prevention (CDC) to monitor the
   prevalence of health-risk youth behaviors. This dataset focuses on
   whether or not youths have recently (in past 30 days) ridden with a
   drunk driver.

   This dataset renamed as YouthRisk for the second edition.

   .. rubric:: Source
      :name: source

   The article "Which Young People Accept a Lift From a Drunk or Drugged
   Driver?" in Accident Analysis and Prevention (July 2009. pp. 703-9)
   provides more details.

   .. rubric:: References
      :name: references

   A more recent version of the full dataset is available at
