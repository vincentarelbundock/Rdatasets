DBH in Schizophrenia
--------------------

Description
~~~~~~~~~~~

This dataset, 'DA_schizophrenia_tbl_df', is a tibble containing
measurements of dopamine ``\beta``-hydroxylase (DBH) activity in 25
schizophrenic patients treated with antipsychotic medication. The data
compares DBH levels between patient groups.

Usage
~~~~~

.. code:: R

   data(DA_schizophrenia_tbl_df)

Format
~~~~~~

A tibble with 25 observations and 2 variables:

dbh
   Integer: Dopamine ``\beta``-hydroxylase activity level
   (nmol/(mL\ ``\cdot``\ hr))

group
   Character: Treatment/patient group classification

Details
~~~~~~~

The dataset name has been changed to ``DA_schizophrenia_tbl_df`` to
provide a shorter, neuroscience-standard abbreviation where "DA" refers
to dopamine. This naming convention maintains clarity and consistency
within the NeuroDataSets package. The suffix ``tbl_df`` indicates that
the dataset is a tibble. The original content has not been modified.

Source
~~~~~~

Data taken from the BSDA package version 1.2.2
