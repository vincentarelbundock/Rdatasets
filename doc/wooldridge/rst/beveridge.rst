      .. rubric:: beveridge
         :name: beveridge

      .. rubric:: Description
         :name: description

      Wooldridge Data loads lazily.

      .. rubric:: Usage
         :name: usage

      ::

         data('beveridge')

      .. rubric:: Format
         :name: format

      A data.frame with 135 observations on 8 variables:

      -  **month:** dec 200 through feb 2012

      -  **urate:** unemployment rate, percent

      -  **vrate:** vacancy rate, percent

      -  **t:** linear time trend

      -  **urate_1:** L.urate

      -  **vrate_1:** L.vrate

      -  **curate:** D.urate

      -  **cvrate:** D.vrate
