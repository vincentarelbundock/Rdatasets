      ======= ===============
      hprice3 R Documentation
      ======= ===============

      .. rubric:: hprice3
         :name: hprice3

      .. rubric:: Description
         :name: description

      Wooldridge Data loads lazily.

      .. rubric:: Usage
         :name: usage

      .. code:: R

         data('hprice3')

      .. rubric:: Format
         :name: format

      A data.frame with 321 observations on 19 variables:

      -  **year:** 1978, 1981

      -  **age:** age of house

      -  **agesq:** age^2

      -  **nbh:** neighborhood, 1-6

      -  **cbd:** dist. to cent. bus. dstrct, ft.

      -  **inst:** dist. to interstate, ft.

      -  **linst:** log(inst)

      -  **price:** selling price

      -  **rooms:** # rooms in house

      -  **area:** square footage of house

      -  **land:** square footage lot

      -  **baths:** # bathrooms

      -  **dist:** dist. from house to incin., ft.

      -  **ldist:** log(dist)

      -  **lprice:** log(price)

      -  **y81:** =1 if year = 1981

      -  **larea:** log(area)

      -  **lland:** log(land)

      -  **linstsq:** linst^2
