Exposures for Vaccine
---------------------

Description
~~~~~~~~~~~

An example SDTM EX dataset for vaccine studies

Usage
~~~~~

.. code:: R

   ex_vaccine

Format
~~~~~~

A data frame with 19 columns:

STUDYID
   Study Identifier

DOMAIN
   Domain Abbreviation

USUBJID
   Unique Subject Identifier

EXSEQ
   Sequence Number

EXLNKGRP
   Link Group ID

EXLNKID
   Link ID

EXTRT
   Name of Actual Treatment

EXCAT
   Category of Treatment

EXDOSE
   Dose per Administration

EXDOSU
   Dose Units

EXDOSFRM
   Dose Form

EXROUTE
   Route of Administration

EXLOC
   Location of Dose Administration

EXLAT
   Laterality

VISITNUM
   Visit Number

VISIT
   Visit Name

EPOCH
   Epoch

EXSTDTC
   Start Date/Time of Treatment

EXENDTC
   End Date/Time of Treatment
