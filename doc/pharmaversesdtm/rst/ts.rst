Trial Design
------------

Description
~~~~~~~~~~~

A SDTM TS dataset from the CDISC pilot project

Usage
~~~~~

.. code:: R

   ts

Format
~~~~~~

A data frame with 6 columns:

STUDYID
   Study Identifier

DOMAIN
   Domain Abbreviation

TSSEQ
   Sequence Number

TSPARMCD
   Trial Summary Parameter Short Name

TSPARM
   Trial Summary Parameter

TSVAL
   Parameter Value
