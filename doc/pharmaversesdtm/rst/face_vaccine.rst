Findings About Clinical Events for Vaccine
------------------------------------------

Description
~~~~~~~~~~~

An example SDTM FACE for vaccine studies

Usage
~~~~~

.. code:: R

   face_vaccine

Format
~~~~~~

A data frame with 30 columns:

STUDYID
   Study Identifier

DOMAIN
   Domain Abbreviation

USUBJID
   Unique Subject Identifier

FASEQ
   Sequence Number

FALNKGRP
   Link Group ID

FALAT
   Laterality

FALNKID
   Link ID

FALOC
   Location of the Finding About

FATESTCD
   Findings About Test Short Name

FATEST
   Findings About Test Name

FAOBJ
   Object of the Observation

FACAT
   Category for Findings About

FASCAT
   Subcategory for Findings About

FAEVAL
   Evaluator

FAORRES
   Result or Finding in Original Units

FAORRESU
   Original Units

EPOCH
   Epoch

FADTC
   Date/Time of Collection

FADY
   Study Day of Collection

FATPT
   Planned Time Point Name

FATPTNUM
   Planned Time Point Number

FATPTREF
   Time Point Reference

FARFTDTC
   Date/Time of Reference Time Point

FAEVLINT
   Evaluation Interval

FAEVINTX
   Evaluation Interval Text

FASTAT
   Completion Status

FAREASND
   Reason Not Performed

FASTRESC
   Character Result/Finding in Std Format

FASTRESN
   Numeric Result/Finding in Standard Units

FASTRESU
   Standard Units

Details
~~~~~~~

Findings About Clinical Events for Vaccine
