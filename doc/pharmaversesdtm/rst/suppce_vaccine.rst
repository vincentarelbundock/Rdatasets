Supplemental Qualifiers for Clinical Events for Vaccine
-------------------------------------------------------

Description
~~~~~~~~~~~

An example SDTM SUPPCE for vaccine studies

Usage
~~~~~

.. code:: R

   suppce_vaccine

Format
~~~~~~

A data frame with 9 columns:

STUDYID
   Study Identifier

USUBJID
   Unique Subject Identifier

RDOMAIN
   Related Domain Abbreviation

IDVAR
   Identifying Variable

IDVARVAL
   Identifying Variable Value

QNAM
   Qualifier Variable Name

QVAL
   Data Value

QLABEL
   Qualifier Variable Label

QORIG
   Origin
