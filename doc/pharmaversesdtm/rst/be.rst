Biospecimen Events
------------------

Description
~~~~~~~~~~~

A synthetic SDTM BE domain representing specimen collection, aliquoting,
and culturing events, with linkage to MB and MS domains

Usage
~~~~~

.. code:: R

   be

Format
~~~~~~

A data frame with 13 columns:

STUDYID
   Study Identifier

DOMAIN
   Domain Abbreviation

USUBJID
   Unique Subject Identifier

BESEQ
   Sequence Number

BEREFID
   Reference ID

BELNKID
   Link Identifier

BETERM
   Reported Term for the Biospecimen Event

BECAT
   Category for Biospecimen Event

BELOC
   Anatomical Location of Event

VISITNUM
   Visit Number

BEDTC
   Date/Time of Specimen Collection

BESTDTC
   Start Date/Time of Biospecimen Event

BEENDTC
   End Date/Time of Biospecimen Event

Details
~~~~~~~

Biospecimen Events
