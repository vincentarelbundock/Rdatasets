Supplemental Qualifiers for Findings About for Clinical Events for Vaccine
--------------------------------------------------------------------------

Description
~~~~~~~~~~~

An example SDTM SUPPFACE dataset for vaccine studies

Usage
~~~~~

.. code:: R

   suppface_vaccine

Format
~~~~~~

A data frame with 9 columns:

STUDYID
   Study Identifier

USUBJID
   Unique Subject Identifier

RDOMAIN
   Related Domain Abbreviation

IDVAR
   Identifying Variable

IDVARVAL
   Identifying Variable Value

QNAM
   Qualifier Variable Name

QVAL
   Data Value

QLABEL
   Qualifier Variable Label

QORIG
   Origin
