Supplemental Qualifiers for Exposures for Vaccine
-------------------------------------------------

Description
~~~~~~~~~~~

An example SDTM SUPPEX dataset for vaccine studies

Usage
~~~~~

.. code:: R

   suppex_vaccine

Format
~~~~~~

A data frame with 9 columns:

STUDYID
   Study Identifier

USUBJID
   Unique Subject Identifier

RDOMAIN
   Related Domain Abbreviation

IDVAR
   Identifying Variable

IDVARVAL
   Identifying Variable Value

QNAM
   Qualifier Variable Name

QVAL
   Data Value

QLABEL
   Qualifier Variable Label

QORIG
   Origin
