      ============== ===============
      hepatoCellular R Documentation
      ============== ===============

      .. rubric:: hepatoCellular
         :name: hepatocellular

      .. rubric:: Description
         :name: description

      Overall and recurrence-free survival of patients with
      hepatocellular carcinoma.

      .. rubric:: Usage
         :name: usage

      ::

         data("hepatoCellular")

      .. rubric:: Format
         :name: format

      A data frame with 227 observations on 48 clinical and biomarker
      variables

      ``Number``
         Patient ID number

      ``Age``
         a numeric vector

         a numeric vector
