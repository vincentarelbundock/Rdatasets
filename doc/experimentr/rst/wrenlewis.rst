      A data frame with 102 rows and 3 variables:

      treated
         Treatment indicator. 1 = treatment, 0 = control

      tcl_area_L_from2010to2017
         Hectares of forest loss 2010-2017

      tcl_area_L_from2001to2009
         Hectares of forest loss 2001-2009
